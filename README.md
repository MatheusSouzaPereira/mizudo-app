# Mizudo-app
Aplicação para gerenciamento de membros da associação de Karatê

## Tecnologias utilizadas : 

- Java 18 
- Quarkus 
- JAX-RS
- Autenticação com JWT
- Angular 13 
- Ionic v6 (em estudos)
- PostgresSQL
 
## Hospedagem

- Heroku
